# RisuAI Plugin Examples

`plugin_start.7z` is the official RisuAI TypeScript plugin starter archive copied from `public/plugin_start.7z` in the upstream repository.

The archive is kept intact because this environment does not include a 7-Zip-compatible extractor. To inspect or use it, extract it with 7-Zip or another tool that supports `.7z`/LZMA.

For inline official JavaScript examples, see `docs/risuai/plugins.md` under the `Examples`, `TTS Hooks`, `Custom AI Provider`, and `MCP Integration` sections. Verify snippets against `docs/risuai/types/risuai.d.ts` before reusing them.
